class pru.Calculadora¶

Bases: object

Una simple calculadora que realiza operaciones básicas.

Métodos: - sumar(a, b): Retorna la suma de dos números. - restar(a, b): Retorna la resta de dos números.

Ejemplo: >>> calc = Calculadora() >>> calc.sumar(2, 3) 5 >>> calc.restar(5, 3) 2

sumar(a: float, b: float) → float¶

Retorna la suma de dos números.

Parámetros:

  • a – El primer número (float).

  • b – El segundo número (float).

Devuelve:

La suma de a y b (float).
